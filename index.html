MATY TRANSPORT VOYAGEUR

Description : Conception d'une application de gestion de tickets de voyage en Visual Basic avec une base de données Access.
Fonctionnalités :
- Enregistrement des passagers (Nom, Prénom, Départ, Destination).
- Choix entre aller simple ou aller-retour.
- Calcul automatique du montant en fonction du nombre de personnes.
Exemple :
Départ : Libreville
Destination : Franceville
Type : Aller-retour
Nombre de personnes : 4
Montant : 120000 XAF
